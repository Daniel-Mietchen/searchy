About Searchy

Searchy is a Wikidata-powered, semantic search engine.

Dashboard visualizations are rendered via wbib ("wah-bib").

Source code for the application is available at the GitHub repository

Searchy is a part of a series of projects related to Wikidata-based management of scientific literature information. Other projects include:

  • Wikidata Bib - A bibliography management tool based on Wikidata (source)
  • ANN - A tool to annotate scientific text with Wikidata IDs (source)
  • Topic Tagger - Adding keywords to articles on Wikidata (source)
